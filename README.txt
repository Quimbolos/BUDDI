3rd Year project as an Aerospace Engineer at University of Bristol, that uses a Matlab Code to perfom the VTOL Analysis of an agile fixed-wing UAV (BUDDI)

This project was part of another project : CASCADE Open Aircraft Project: University of Bristol VTOL Drone Development, which was published by the AIAA (American Institute of Aeronautics and Astronautics)

Published paper: https://arc.aiaa.org/doi/10.2514/6.2021-1930
